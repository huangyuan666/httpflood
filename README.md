# httpflood
An HTTP Flooding tool written with Python 3
## Required Python Modules:
```
scapy, requests
```
### Installation:
```
pip3 install scapy
pip3 install requests
```
## Installation:
```
cd
git clone https://github.com/ryland192000/httpflood
cd httpflood
```
## Usage:
```
python3 httpflood.py -u <url> -p <port> -r <number of requests>
```
